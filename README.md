# Student-Record-Management-System

📌 Project Overview

The Student Record Management System is a software application designed to manage and organize student information efficiently. It helps in storing, updating, retrieving, and deleting student records in a structured manner.

This system reduces manual paperwork and provides a simple interface to manage student data such as personal details, academic information, and course records.

🎯 Features

Add new student records

View student details

Update existing student information

Delete student records

Search for a specific student

Store and manage data efficiently

🛠️ Technologies Used

Programming Language: Python

Database: SQL / SQLite

Tools: VS Code, GitHub

Concepts Used: CRUD Operations, Database Management

⚙️ How It Works

The system connects to the database.

Users can perform operations such as adding, viewing, updating, and deleting student records.

The data is stored securely in the database.

Users can retrieve student information anytime.

▶️ How to Run the Project
